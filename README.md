# 🎒 실패 없는 취미 생활을 위한 장비 공유 및 커뮤니티 플랫폼 "장비빨"
<img width="255" height="118" alt="image" src="https://github.com/user-attachments/assets/ee147b2a-da9f-4469-a15e-d3d436115a2c" />

## 프로젝트 소개
> 장비빨은 취미 진입 장벽을 낮추기 위한 라이프스타일 커뮤니티 서비스입니다.<br/>
>현대인들은 새로운 취미를 시작하고 싶어 하지만, 수많은 장비 정보 속에서 선택의 피로를 느낍니다. <br/>
> 장비빨은 취미를 즐기는 유저들의 장비 조합과 생생한 활동 피드를 통해 확실한 구매 가이드를 제공할 수 있도록 합니다.
